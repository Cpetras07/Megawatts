Quem somos?

Fazemos parte de um projeto da Universidede Paulista (UNIP), do curso de Ciência da Computação.
Este projeto pertence à matéria de IHC (Interface Humano-Computador), que tem como objetivo principal fornecer aos desenvolvedores de sistemas explicações dos fenômenos de interação usuário-sistema, tendo resultados práticos para o design de uma interface de usuário.

O tema do trabalho é sustentabilidade, porém decidimos nos aprofundar no subtema: Energia.
Demos o nome ao nosso grupo de Megawatt's!

Proposta:

A proposta feita foi: criar um canal no YouTube para abordar, explicar e discutir o tema escolhido. Com o objetivo de bater uma meta de inscritos e visualizações, postamos vídeos semanalmente. Estes vídeos foram feitos de uma forma descontraída e bem dinâmica, para que o público entendesse de um jeito diferente e didático o assunto tratado.

Para concluir este desafio, tivemos que separar os 5 integrantes do grupo em diferentes funções. São eles:
